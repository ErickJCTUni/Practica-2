Numero Etiqueta Descripcion Uso
1 <html> Define la raiz de un documento HTML
2<!--...-->Define un comentario
3<!DOCTYPE>Define el tipo de documento que se va a crear
4<a>Utilizado para crear hyperlinks (enlaces)
5<article>Utilizado para crear separaciones en el documento HTML y que el navegador entienda que el bloque introducido sera un articulo
6<audio>Permite insertar audio al documento HTML
7<b>Intoduce el efecto "bold" en el texto dentro del documento HTML
8<body>Etiqueta que define el cuerpo del documento
9<br>Esta etiqueta permite un salto de linea en el documento HTML
10<button>Etiqueta que crea un documento
